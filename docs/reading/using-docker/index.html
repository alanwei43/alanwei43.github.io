Docker开发指南

常用Docker命令:

启动一个容器#

基本命令: docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

选项 Options#

--name 指定容器名称(后续其他docker命令可以使用) -h, --hostname 指定容器 hostname:

docker run --name container_name -h host_name debian echo "hello world"

-i 进入交互式模式 -t 容器终端输出附加到当前终端

启动一个debian容器, 并进入交互式模式(-i), 在终端(-t)执行执行 /bin/bash:

docker run -i -t debian /bin/bash

--rm 退出容器后自动删除容器

查看容器信息#

基本命令: docker inspect [OPTIONS] NAME|ID [NAME|ID...]

# 查看分配给指定容器的IP
docker inspect dfc7f092a6c4 | grep IPAddress

容器日志查询: docker logs [OPTIONS] CONTAINER

docker logs container_name
