yargs​

安装依赖:

npm i yargs@16.2.0 --save

创建入口文件：

bin/cli.ts
#!/usr/bin/env node

import yargs from "yargs";

const ext = __filename.split(".")[__filename.split(".").length - 1] || "js";

yargs.commandDir("command-files-directory", { 
  recurse: true,
  extensions: [ext]
}).demandCommand()
  .version("version", "显示版本号", "0.0.1")
  .help("help", "显示帮助")
  .argv;

创建命令模板:
