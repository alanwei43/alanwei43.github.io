我的Vim基本配置:

配置#

Windows配置文件为 ~/_vimrc, Unix配置文件为 ~/.vimrc

配置内容为:

.vimrc
"colo koehler
filetype plugin on
set rnu
set nu
set nocompatible
set noswapfile
set noswf
set encoding=utf-8
set fileencoding=utf-8
set autoindent
set showcmd
set shiftwidth=4
set tabstop=4
set softtabstop=4

其他#

Vim Ctrl + S#

  • Pressing "Ctrl + S" by mistake while using Vim
  • Vim Trick: Map CTRL+S to Save

解释

当你在vim使用ctrl+s快捷键之后, vim好像被锁定了, 除非你按下ctrl+q退出该模式. 这个叫做 Software flow control.

禁用

编辑~/.bashrc增加一行stty -ixon即可禁用vim的ctrl+s功能.

改造成文件保存

禁用ctrl+s功能之后, 修改vim配置文件(.vimrc), 可以把vim的ctrl+s快捷键改成保存当前文件:

.vimrc
" normal mode
nnoremap <silent><c-s> :<c-u>update<cr>
" visual mode
vnoremap <silent><c-s> <c-c>:update<cr>gv
" insert mode
inoremap <silent><c-s> <c-o>:update<cr>
